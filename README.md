# 🚀 P1Fono — Plataforma Web

Recentemente finalizei melhorias e atualizações no projeto **P1Fono**

O projeto consiste em uma aplicação web desenvolvida com foco em organização, acessibilidade e experiência do usuário, trazendo uma interface moderna e funcional para gerenciamento e apresentação de informações relacionadas à área de fonoaudiologia.

Durante o desenvolvimento, foram utilizadas tecnologias como **HTML, CSS e JavaScript**, buscando criar uma aplicação mais intuitiva, responsiva e eficiente.

---

# 🔧 Principais melhorias implementadas

* ✅ Estruturação mais organizada do código
* ✅ Interface mais moderna e agradável visualmente
* ✅ Melhor separação de responsabilidades entre arquivos
* ✅ Otimização da experiência do usuário
* ✅ Código mais limpo e fácil de manter
* ✅ Layout responsivo para diferentes dispositivos
* ✅ Aplicação de boas práticas de desenvolvimento web

# ▶️ Como executar o projeto localmente

Siga o passo a passo abaixo para rodar o projeto na sua máquina:

## 1️⃣ Clone o repositório

```bash id="3cv5gq"
git clone https://github.com/maryreginaf15/p1fono-maryreginaf.git
```

---

## 2️⃣ Acesse a pasta do projeto

```bash id="6fj3da"
cd p1fono-maryreginaf
```

---

## 3️⃣ Abra o projeto

Você pode abrir o projeto utilizando:

* VS Code
* IntelliJ
* Sublime Text
* Ou qualquer editor de sua preferência

---

## 4️⃣ Execute o projeto

Como o projeto utiliza HTML, CSS e JavaScript puro, basta abrir o arquivo:

```bash id="y4nq1r"
index.html
```

Ou utilizar a extensão **Live Server** no VS Code para uma melhor experiência.

---

# ⚡ Executando com Live Server (Recomendado)

## Instale a extensão:

* Live Server — VS Code Extension

## Depois:

1. Clique com o botão direito no arquivo `index.html`
2. Selecione:

```bash id="b3zw0v"
Open with Live Server
```

O projeto será iniciado automaticamente no navegador.
